# 최종 프로젝트 - 취미 기반 소개팅 플랫폼

## 프로젝트 제목 - 취밋 ( 취미 + meet )  



### 개요

- 취미를 기반으로 이상형을 매칭해서 소개를 받고, 미팅에 참석하는 등 다양한 활동을 할 수 있는 소개팅 플랫폼.

### 주요 기능

1. 로그인 / 회원가입 기능

2. 이상형 연결 기능

3. 호감, 매력도 관리 기능

4. 공지사항 / 이벤트 관리 기능

5. 미팅 관리 기능

6. 채팅 기능

7. 화상 채팅 기능

8. 문의사항 관리 기능

9. 유저 정보 관리 기능 ( 프로필 / 계정 정보 )

10. 실시간 알림 기능

11. 썸네일 관리 기능

12. 랜덤 이상형 탐색 기능 <br />
